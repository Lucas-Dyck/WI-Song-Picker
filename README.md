# WI-Song-Picker
A lightweight program that picks randomly from a library of Worship Initiative worship songs. Use this program to boost your worship instrument practice, or to broaden your worship song horizons, and play worship songs you've never played before.

This program is catered towards the Worship Initiative song library, but any music software website like Multitracks.com, iTunes, Music, or Spotify, works (note that the url link will only work for the Worship Initiative.com).
