# WI-Song-Picker
A lightweight program that picks randomly from a library of Worship Initiative worship songs. Use this program to boost your worship practice time, or to broaden your worship song repertoire, and learn new songs.

This program is catered towards the Worship Initiative song library, but any site like Multitracks.com, iTunes, Music, or Spotify, can be used (note that the url link will only work for the Worship Initiative.com).
